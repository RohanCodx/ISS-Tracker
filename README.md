# 🚀 ISS Tracker 2D

A simple and lightweight **2D ISS (International Space Station) Tracker** that shows the real-time position of the ISS on a world map 🌍.

---

## 📌 Features

* 🌎 Real-time ISS location tracking
* ⚡ Smooth and lightweight performance
* 🗺️ 2D map-based visualization
* 📱 Works on both desktop and mobile browsers
* 🔄 Auto-updating position

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## ⚙️ How It Works

The project fetches real-time ISS location data using a public API and updates its position on the map dynamically.

---

## 📥 Installation / Usage

1. Download or clone this repository
2. Open `index.html` in your browser
